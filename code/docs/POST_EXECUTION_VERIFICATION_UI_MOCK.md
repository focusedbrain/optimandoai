### Clear Disclaimers

| Location | Disclaimer |
|----------|------------|
| Page header | PoAE status notice (full banner) |
| Each [DEMO] badge | Tooltip: "Placeholder â€” not implemented" |
| Export dialog | Warning if demo data included |

### Audit-Friendly Features

| Feature | Purpose |
|---------|---------|
| Hash visibility | All hashes copyable for external verification |
| Chain integrity check | One-click verification of audit chain |
| Policy snapshot | Immutable record of rules at execution time |
| Export without demo | Clean export for legal/compliance use |

### What NOT to Claim

| Avoid | Why |
|-------|-----|
| "Cryptographically proven" | PoAE not implemented |
| "Tamper-proof" | Only audit chain is verified, not full PoAE |
| "Legally binding" | Requires legal review per jurisdiction |
| "Attestation verified" | No attestation exists yet |
