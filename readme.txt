📘 Paper v201: The Official Canon
The foundation of this repository is the cornerstone scientific and normative work: "WR Desk™ – Browser-Based Multi-Agent Orchestration for Contextual Workflow Automation" (v201).

Starting with v191, this document serves as the sole authoritative source (The Canon) for the following protocols:

WR Desk™ (Workflow Ready Desk): The unified framework and publishing standard for template-driven agent logic.

Wr Code® (Workflow Ready Code): The unified Transport layer of Automation Workflows 

BEAP™ (Bidirectional Email Automation Protocol): The standard for secure, two-way automated workflow handling.

PoAE™ (Proof of Authoritative Execution): The cryptographic verification layer ensuring intent-bound execution.
